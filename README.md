### Getting Started
Для локального запуска
1. Клонирование репозитория
```sh
git clone https://github.com/Ankrait/sea-battle_back.git
```
2. Установка зависимостей
```sh
yarn install

// or with npm

npm install
```
3. Старт сервера
```sh
yarn start

// or with npm

npm start
```
